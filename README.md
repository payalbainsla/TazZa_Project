# Tazza 🍓🥬

Tazza is a fresh fruits and vegetables web app where users can browse and buy organic produce.  
Built using **HTML**, **CSS**, **Bootstrap**, and **JavaScript**.

## 🌟 Features
- Responsive layout
- Product cards
- Smooth navigation

## 🛠️ Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
